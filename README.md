## Instalamos jest:

`npm install --save-dev jest`


## Agregamos o modificamos el package.json

`
{
 "scripts": {
   "test": "jest"
 }
}`


## Ejecutamos los tests:

`npm run test`

## Con coverage:

`npm run test -- --coverage`

## Podemos ver el coverage en un html ingresando a:

`coverage/lcov-report/index.html`
